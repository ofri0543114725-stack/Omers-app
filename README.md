# עיצוב מסמכים אוטומטי

אפליקציה להעלאת קובץ Word ועיצובו אוטומטית לפי תקן קבוע.

## כללי העיצוב
- **עברית**: David גודל 13
- **אנגלית**: Times New Roman גודל 11
- **בולטים**: הופכים ל- `-` מיושרים לימין
- **יישור**: ימין (בטבלאות — מרכז)
- **רווח בין פסקאות**: שורה ריקה David גודל 11
- **רווח שורות**: 1.0
- **דף**: A4

## הרצה מקומית

```bash
pip install -r requirements.txt
streamlit run app.py
```

## פריסה ב-Streamlit Cloud

1. העלה את הקוד ל-GitHub (repository חדש)
2. כנס ל-[share.streamlit.io](https://share.streamlit.io)
3. חבר את ה-GitHub repository
4. בחר `app.py` כקובץ הראשי
5. לחץ Deploy
